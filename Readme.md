# Python Basics Assignment

## Overview
This repository contains solutions to 20 basic Python programming problems. The exercises cover fundamental Python concepts such as variables, data types, operators, strings, user input, conditional statements, and simple problem-solving.

## Topics Covered

- Variables and Data Types
- Type Conversion
- Multiple Assignment and Variable Swapping
- Dynamic Typing
- Boolean Conversion
- Arithmetic Operators
- Assignment Operators
- Comparison Operators (`==` vs `is`)
- Logical Operators
- Membership Operators
- String Methods
- Split and Join
- String Slicing
- String Searching and Validation
- String Formatting
- User Input
- Rectangle Area & Perimeter Calculator
- Grade Calculator
- Simple Login System
- Positive/Negative/Zero and Even/Odd Checker

## File Structure

```
.
├── main.py        # Python source code containing all 20 problems
└── README.md      # Project documentation
```

## Requirements

- Python 3.x

## How to Run

1. Clone the repository.
2. Open the project folder.
3. Run the program using:

```bash
python main.py
```

or

```bash
python3 main.py
```

## Note

Some problems require user input. Enter the requested values when prompted in the terminal.
